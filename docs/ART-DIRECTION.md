# Tactical Duelist — Art Direction & Moodboards

## Общий стиль

**Одно предложение:** Brawl Stars встречает Into The Breach в 3D — 
яркий, читаемый, тактический, с характером.

### Ключевые принципы

1. **Читаемость прежде всего.** Игрок смотрит на сетку сверху-вниз под углом. 
   Каждый герой, стена, тайл должны быть мгновенно различимы.
   Силуэт героя = его идентичность. Даже без текстур — по форме понятно кто есть кто.

2. **Стилизованный, НЕ реалистичный.** Low-poly stylized. 
   Чистые формы, яркие цвета, минимум мелких деталей.
   Ближе к Brawl Stars / Clash Mini / Archero, НЕ к Genshin / PUBG.

3. **Контрастные цвета.** Каждый герой = уникальная цветовая палитра.
   Фон (пол/стены) — нейтральный. Герои — насыщенные.
   UI — чёткий, высокий контраст, крупные элементы (мобильные пальцы!).

4. **Тактическая атмосфера.** Не мультяшный хаос (как Brawl Stars), 
   а сдержанная тактическая уверенность. Ближе к шахматам с пушками.
   Герои выглядят серьёзно, но стилизованно.

## Мудборд #1: Общий визуальный стиль

### Референсы для 3D-артиста

**Игры-референсы (визуальный стиль):**
- Brawl Stars — уровень детализации персонажей, изометрический вид
- Clash Mini — 3D фигурки на сетке, чистые формы
- Into The Breach — тактическая сетка, читаемость, UI поверх поля
- Archero — stylized 3D герои, яркие VFX
- Auto Chess / Teamfight Tactics — 3D модели на игровом поле
- Slay The Spire — UI карт действий (для планирования)
- Minit — минималистичность, фокус на механике

**Пропорции персонажей:**
- Chibi / SD (super-deformed): голова = 1/3 тела
- 2.5-3 головы высотой (не реалистичные 7-8 голов)
- Крупные руки/оружие для читаемости силуэта
- Гипертрофированные ключевые элементы (лук Лучника ОГРОМНЫЙ)

**Визуальная цель:**
```
Реализм ←——————|—X———→ Мультяшность
                    ^
                Мы здесь:
                Стилизованный, но с характером.
                Не kiddie, не gritty. "Smart cartoon."
```

## Мудборд #2: Герои — силуэты и характеры

### Каждый герой должен быть узнаваем по силуэту

```
ARCHER (Лучник)
  Силуэт: стройный, вертикальный, лук торчит вверх/вбок
  Палитра: зелёный + коричневый (лесной рейнджер)
  Ключевой элемент: ОГРОМНЫЙ лук или арбалет
  Характер: спокойный, расчётливый, Hood vibes
  Поза idle: лук опущен, стрела наготове

TANK (Танк)
  Силуэт: ШИРОКИЙ, приземистый, квадратный
  Палитра: серый + оранжевый (металл + огонь)
  Ключевой элемент: массивный щит в одной руке, короткий ствол в другой
  Характер: несокрушимый, уверенный, медленный
  Поза idle: щит перед собой, слегка пригнут

SHADOW (Тень)
  Силуэт: острый, угловатый, плащ/шарф развевается
  Палитра: тёмно-фиолетовый + чёрный + cyan акценты
  Ключевой элемент: два коротких клинка, маска на лице
  Характер: быстрый, опасный, ninja vibes
  Поза idle: низкая стойка, готов к рывку

SCOUT (Разведчик)
  Силуэт: средний, гаджеты на поясе, антенна/сканер
  Палитра: синий + белый (tech/intel)
  Ключевой элемент: визор/сканер на глазу, наплечный гаджет
  Характер: умный, быстрый, информированный
  Поза idle: смотрит в сканер, расслаблен но внимателен

MAGE (Маг)
  Силуэт: мантия расширяется книзу, посох выше головы
  Палитра: фиолетовый + золотой (магия + мудрость)
  Ключевой элемент: светящийся посох с кристаллом
  Характер: мудрый, отстранённый, мощный
  Поза idle: левитирует на 10см над землёй, мантия парит

DEMO (Подрывник)
  Силуэт: коренастый, рюкзак за спиной ОГРОМНЫЙ
  Палитра: красный + жёлтый (danger/explosion)
  Ключевой элемент: рюкзак с бомбами, детонатор в руке
  Характер: весёлый маньяк, любит взрывы
  Поза idle: подкидывает бомбу одной рукой

GUARDIAN (Страж)
  Силуэт: высокий, массивный, щит-барьер перед собой (энергетический)
  Палитра: голубой + белый + серебро (защита, свет)
  Ключевой элемент: генерируемый энергощит (полупрозрачный)
  Характер: непоколебимый защитник, паладин vibes
  Поза idle: рука на генераторе щита, спокоен

GHOST (Призрак)
  Силуэт: вытянутый вверх, ПАРИТ, нет ног (или полупрозрачные ноги)
  Палитра: бледно-голубой + белый + прозрачность
  Ключевой элемент: вместо ног — дымка/шлейф, пустые светящиеся глаза
  Характер: жуткий но не злой, потусторонний
  Поза idle: медленно покачивается, частицы отлетают

ENGINEER (Инженер)
  Силуэт: средний, МНОГО деталей на теле (инструменты, провода)
  Палитра: жёлтый + серый (construction/tech)
  Ключевой элемент: дрон/мини-турель на плече или за спиной
  Характер: nerd энтузиаст, всегда что-то мастерит
  Поза idle: крутит гаечный ключ, дрон кружит вокруг

BERSERKER (Берсерк)
  Силуэт: ОГРОМНЫЙ, самый большой, два топора по бокам
  Палитра: красный + тёмно-серый (ярость + сталь)
  Ключевой элемент: два огромных топора, минимум брони, шрамы
  Характер: неудержимая ярость, но не тупой — контролируемая агрессия
  Поза idle: тяжело дышит, топоры покачиваются

HAWK (Ястреб)
  Силуэт: длинный, тонкий, винтовка = самый длинный элемент
  Палитра: коричневый + тёмно-зелёный (маскировка/снайпер)
  Ключевой элемент: длиннющая снайперская винтовка, прицел-линза
  Характер: терпеливый, один выстрел — один kill
  Поза idle: смотрит в прицел, неподвижен

MIRAGE (Мираж)
  Силуэт: нормальный, но ДВОИТСЯ (лёгкий ghost-эффект вокруг)
  Палитра: бирюзовый + розовый + зеркальные отблески
  Ключевой элемент: зеркальные/голографические элементы на костюме
  Характер: загадочный, трикстер, фокусник
  Поза idle: силуэт слегка "рябит" как голограмма
```

## Мудборд #3: Игровое поле (арена)

### Ракурс камеры

```
- Изометрический 3D вид (Perspective projection)
- Угол: ~45° сверху-вниз (стиль Brawl Stars)
- Камера может свободно вращаться вокруг поля (опционально для MVP)
- Zoom: достаточный чтобы видеть всю сетку на мобильном экране
```

### Стиль тайлов

```
MVP — Arena Theme:
- Пол: плоский, слегка текстурированный (песчаник/камень)
- Стены: 3D блоки, ~1.5 высоты тайла, стилизованные
- Края карты: пропасть / обрыв / декоративная граница
- Освещение: яркое, directional light ~45°, мягкие тени (URP)
- Нет тумана, нет темноты — всё должно быть ЧИТАЕМО

Вдохновение:
- Brawl Stars арены
- Clash Royale арена
- Auto Chess доски
- Chess.com 3D доска
```

### Подсветка тайлов (КРИТИЧНО для геймплея)

```
Фаза планирования:
- Доступные клетки для движения = зелёная полупрозрачная подсветка
- Линия выстрела = красная полупрозрачная полоса по направлению
- Текущая позиция героя = яркий контур вокруг тайла
- Позиция противника = (если видна) контур другого цвета

Фаза выполнения:
- Текущий шаг = подсвечиваемые тайлы движения/выстрела
- Trail движения = затухающий след (куда герой прошёл)
- Попадание = яркая вспышка на тайле
```

## Мудборд #4: UI стиль

### Общие принципы UI

```
- Mobile-first: кнопки минимум 44×44px tap area
- Толстые контуры (2-4px borders на элементах)
- Скруглённые углы (8-16px radius)
- Drop shadows на панелях (глубина)
- Цветовая схема: тёмный фон + яркие акценты
- Шрифт: bold, без засечек, крупный (16px минимум)
- Вдохновение: Brawl Stars UI, Clash Royale UI, Supercell стиль
```

### Планирование — Action Queue UI

```
- Горизонтальная полоса слотов внизу экрана (как колода карт)
- Каждый слот = квадрат с иконкой действия
- Пустые слоты = пунктирная рамка
- Заполненные = яркая иконка + номер шага
- Кнопки действий = крупные иконки над слотами
- Кнопка Shoot = серая когда на кулдауне (с числом CD)
- Кнопка Undo = в углу
- Кнопка Confirm = большая зелёная внизу-справа
- Таймер = крупные цифры сверху-по центру

Вдохновение:
- Into The Breach: выбор действий
- Slay The Spire: карты в руке
- Clash Royale: карты внизу экрана
```

## Мудборд #5: Экран выполнения (Execution)

```
Во время execution фазы:
- UI минимален — только grid + герои + VFX
- Прогресс-бар шагов сверху (step 1/6, 2/6...)
- Кнопки скорости: 1x, 2x, пошагово
- Каждый шаг анимируется ~0.8с (1x)
- Камера может слегка следить за действием (динамический zoom)
- Попадание = замедление (0.3s slowmo) + screen shake + вспышка
- Mutual cancel = особая визуализация (два луча сталкиваются в центре)
- Kill = zoom на жертву, замедление, explosion, победный sting
```

## Для 3D артиста: чеклист на старт

### Что нужно сделать ПЕРВЫМ

```
1. КОНЦЕПТ-АРТЫ (2D скетчи):
   □ 4 героя MVP: Archer, Tank, Shadow, Scout
   □ По 3 варианта каждого (итого 12 скетчей)
   □ Turnaround (фронт, профиль, спина) для утверждённого варианта
   □ Color palette для каждого героя

2. ТАЙЛОВЫЙ СЕТ (3D):
   □ Floor tile (1 unit × 1 unit, UV mapped)
   □ Wall block (1 × 1 × 1.5, стилизованный)
   □ Grid lines overlay (тонкие линии между тайлами)
   □ Протестировать в Unity: 10×10 сетка выглядит хорошо?

3. ПЕРВЫЙ ГЕРОЙ (3D, полный пайплайн):
   □ Выбрать одного героя (рекомендую Archer — нейтральный дизайн)
   □ Моделирование (3000-5000 tri)
   □ UV + текстура (512×512)
   □ Риг (Unity Humanoid)
   □ Базовые анимации: Idle, Walk, Shoot, Death
   □ Импорт в Unity, проверить URP Lit shader
   □ Протестировать на сетке: читается? виден? красивый?

4. После утверждения пайплайна:
   □ Остальные 3 героя MVP
   □ VFX выстрела (простой particle trail)
   □ VFX попадания (вспышка)
```

### Технические требования для Unity

```
Формат моделей: FBX
Полигонаж: 3000-5000 tri на героя
Текстуры: 512×512 PNG, albedo + normal (опционально)
Риг: Unity Humanoid Avatar
Анимации: FBX, 30fps, отдельные клипы или split в Unity
Шейдер: URP/Lit (Standard Surface для preview)
Масштаб: 1 unit = 1 метр (герой ~1 unit высотой)
Ориентация: Y-up, -Z forward (Unity standard)
```
